netspd_

network speed, beautifully measured — in your terminal

Install

# macOS / Linux (Homebrew)
brew tap TarunVishwakarma1/tap && brew install netspd

# Rust
cargo install netspd

# Nix
nix run github:TarunVishwakarma1/netspd

# Windows / Linux binaries
https://github.com/TarunVishwakarma1/netspd/releases/latest

Run

netspd                          # the full experience
netspd --json | jq              # scripts and pipelines
netspd serve  +  netspd --url http://peer:9516   # pod-to-pod / LAN

Why netspd

  • Hypercar tachometer with spring-physics needle
  • Bufferbloat grade (A+–F) — latency under load
  • Plain-language verdict of what your link can do
  • LibreSpeed, Ookla, Fast.com or your own servers
  • Client and server in one static binary
  • JSON, CSV, Prometheus textfile output
  • History, trends chart, scheduled repeat runs
  • Themes, mouse support, 80×24 → 4K, --ascii
